Robust C/C++ Programming with Valgrind

C and C++ are very powerful languages

But great power comes with great responsability

What is valgrind

Valgrind is not a debugging tool

Valgrind is a dynamic analysis framework

I can help us to detect issues in our program at run-time

It includes multiple built-in tools

  • Memcheck
  • Massif
  • CacheGrind
  • Helgrind
  • ...

What can we do with Valgrind

  • Detect memory management issues
  • Memory profiling
  • Detect threading issues
  • File descriptors not closed
